PERCo Security week – 2019

Производитель оборудования безопасности

C 14 по 18 октября партнеры PERCo из Франции, Бельгии, Сербии и Польши приняли участие в тренинге PERCo Security week, который прошел в Санкт-Петербурге.

Участники прошли техническое обучение работе с оборудованием PERCo в Учебном центре, познакомились с новыми товарами, обсудили дальнейшее развитие сотрудничества и совместного продвижения продукции PERCo.
