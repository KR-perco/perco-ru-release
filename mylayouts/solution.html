Тумбовые турникеты

Удобным решением для объектов платного доступа являются тумбовые турникеты. Универсальной моделью для интеграции с системами платного доступа является тумбовый турникет TTD-10A, модульная конструкция которого позволяет встраивать:

  • сканеры штрихкода
  • биометрические устройства
  • считыватели карт
  • монетоприемники
  • картоприемники

Полноростовые турникеты

Для объектов с повышенными требованиями к безопасности, например, для стадионов и морских портов, оптимальным решением станут полноростовые турникеты:

  • полное перекрытие зоны прохода
  • дополнение полноростовыми калитками для организации эвакуационного выхода
  • возможность работы в режиме шлюза

Для разграничения потоков «вход/выход» устанавливаются калитки, позволяющие организовать свободный проход в одну сторону и запрет прохода в другую.

Скоростные проходы

Для культурных учреждений удобным решением станут скоростные проходы – турникеты с распашными или раздвижными створками:

  • бесконтактный проход
  • максимальная пропускная способность
  • эффектный дизайн
  • простота интеграции с дополнительным оборудованием

Конструкции турникетов могут быть адаптированы в соответствии с задачами объекта. Например, в кинотеатре турникеты для удобства зрителей могут быть дополнены подставкой для напитков и попкорна. Для противодействия распространению инфекции в местах массового пребывания людей система контроля доступа может быть интегрирована с пирометром PERCo. Контроль температуры также может быть реализован с помощью терминалов распознавания лиц, способных, среди прочего, определять факт наличия на лице маски.
