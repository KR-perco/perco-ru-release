Вышла новая версия системы PERCo-Web 2.0

Основные отличительные особенности:

  • Гибкость и универсальность: сервер системы можно установить на Windows и Linux
  • Высокая производительность: реализация на платформе NODE JS, модульная архитектура, поддержка многоядерных процессоров
  • Использование системы управления базами данных MySQL
  • Простота интеграции сторонних приложений за счет реализации функций SDK в формате JSON API
  • Интеграция с 1С
